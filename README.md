# AI Study Assistant 🤖

Full-stack AI chatbot built with React + Express + Groq API to help students study smarter.

### Features
- Real-time AI chat using Groq's Llama 3
- React frontend with clean UI
- Express backend with secure API handling
- CORS enabled for local development

### Tech Stack
**Frontend:** React, Axios  
**Backend:** Node.js, Express, Groq SDK  
**API:** Groq - Llama 3.1 8B Instant
